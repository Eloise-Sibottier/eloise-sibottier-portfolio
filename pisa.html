Résumé rapide

Objectif : expliquer les écarts de scores PISA entre pays et prédire les scores pour pays non testés. Modèle (log PISA) : R² ≈ 0.717. PIB/hab & sous-alimentation sont déterminants.

Compétences mobilisées

  • Collecte & nettoyage (OCDE, Banque mondiale)
  • Exploration & data viz (ggplot2, plotly, cartographie)
  • Économétrie : régressions, interactions, diagnostics (R)
  • Lecture critique d’indicateurs internationaux

Régression linéaire — Modèle final (synthèse)

Forme retenue :

ln_PISA = β0 + β1 * dépense_éducation + β2 * ln(PIB/hab) + β3 * scolarisation_primaire
+ β4 * sous_alimentation + β5 * OCDE + β6 * (scolarisation_maternelle : OCDE) + ε

Points clés :

  • R² ajusté ≈ 0.717 (≈ 74% de variance expliquée).
  • PIB/hab (log) : effet positif robuste.
  • Sous-alimentation : effet négatif significatif.
  • Interaction : la scolarisation maternelle a un effet plus fort hors OCDE.
